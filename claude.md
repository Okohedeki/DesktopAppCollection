# CommonExtractors — Tier 1

A local-first desktop app for high-demand document conversions.
Fast, private, and built for batch workflows.

## Core Tools

### 1. PDF → Excel  
High demand. Used for finance, reports, and data extraction.

### 2. PDF → Word  
Editable documents without retyping or layout breakage.

### 3. PDF → Clean Text  
Removes headers, footers, line breaks, and noise.

### 4. Scan PDF → Searchable PDF  
OCR for scanned documents. Fully offline.

### 5. Invoice / Receipt → JSON  
Structured extraction for accounting and automation.

## Principles
- Local-only (no uploads)
- Batch processing
- Simple UI
- Predictable outputs

## Goal
Replace sketchy web converters with a fast, reliable desktop tool.
